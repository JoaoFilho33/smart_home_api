# SmartHome API

Este projeto é uma API REST que permite a gestão de dispositivos inteligentes em residências ou empresas. Desenvolvido para fins de prática, o projeto aborda a criação de API REST, documentação com OpenAPI e conceitos de arquitetura de software. A API suporta funcionalidades como ativação/desativação de dispositivos, controle de locais e ambientes, e detecção de dispositivos offline.

## Detalhes Técnicos

- **API Rest**: Gerenciamento de dispositivos inteligentes.
- **Persistência**: Armazenamento em memória.
- **Services**: Implemente Command-Query Separation com testes para os mesmos.
- **Deploy**: Render.
- **Testes de Contrato de API**: Postman.
- **Frontend Web CRUD** (Opcional): Crie uma interface de usuário básica.

## Funcionalidades Principais

- Gestão de **Locais** e **Ambientes**.
- Controle de **Dispositivos Inteligentes**.
- Propriedade e ativação/desativação de dispositivos.
- Desligamento em massa de dispositivos.
- Listagem ordenada e detecção de dispositivos offline.
